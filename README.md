### Quick Start for Contributors / Snelstart voor Bijdragers

1. Create a new branch from `development` / Maak een nieuwe branch vanaf `development`
2. Make your changes / Maak je wijzigingen
3. Push to your own branch / Push naar je eigen branch
4. Create a Pull Request to `development` / Maak een Pull Request naar `development`
5. Wait for approval / Wacht op goedkeuring

**Note**: Direct pushes to `development` are not allowed.
